# 🧠 Health AI Dashboard

## 🚀 Overview

The Health AI Dashboard is a modern web-based health-tech application that helps users analyze their **mental health** and **regional health risks** using an AI-assisted backend model.

The system provides instant insights based on user inputs and ensures reliable results using fallback logic when AI is unavailable.

---

## ✨ Features

- 🧠 Mental Health Analyzer (Text + Stress + Sleep + Work Hours)
- 🌍 Regional Health Risk Analysis (State & District based)
- 📊 Stress vs Sleep Visualization
- 🧘 Recovery Protocol Suggestions
- ⚡ AI-powered insights using Gemini API
- 🛡️ Fallback logic (works even without AI)
- 📱 Responsive and Modern UI (Dark Theme + Glassmorphism)

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js (Express)  
- API: Google Gemini (Free Tier - gemini-1.5-flash)  
- Deployment: Local / Free Platforms  
- Approach: AI-assisted full-stack development

## ⚙️ How It Works

1. User enters mental health data (text, stress, sleep, work hours)  
2. User selects location (state & district)  
3. Backend processes data using AI (Gemini API)  
4. If AI fails, fallback logic generates results  
5. System displays:
   - Mood & Burnout Level  
   - Risk Level & Diseases  
   - Personalized Suggestions  

---

## 🧩 System Architecture

User → Frontend UI → Backend API → Gemini AI / Fallback Logic → Response Processing → Result Display
